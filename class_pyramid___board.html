Detailed Description

Represents the pyramid-shaped board.

The internal board is a 3x5 matrix, but only specific cells form the pyramid. Valid positions:

  • Row 0: Col 2
  • Row 1: Cols 1, 2, 3
  • Row 2: Cols 0, 1, 2, 3, 4
