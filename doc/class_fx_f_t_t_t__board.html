int FxFTTT_board::check_all_lines ( )

Counts the total 3-in-a-row patterns for both X and O.

Iterates through Rows, Cols, and Diagonals to count 3-in-a-row patterns.

Returns
1 if X > O, -1 if O > X, 0 if equal.
1 if X wins count, -1 if O wins count, 0 if draw.
